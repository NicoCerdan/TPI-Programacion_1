# Trabajo Integrador – Programación I (1C 2026)

## 📌 Descripción del proyecto
Este proyecto consiste en un programa en Python que permite **gestionar información de países**, aplicando operaciones de búsqueda, filtrado, ordenamiento y estadísticas.  
El objetivo es integrar los conocimientos de programación estructurada, modularización y manejo de excepciones.

## 🎯 Objetivos
- Aplicar estructuras de datos dinámicas (listas y diccionarios).  
- Implementar un menú interactivo con funciones modulares.  
- Incorporar validaciones y manejo de errores (`try/except`).  
- Documentar el código y elaborar un informe académico.
